Operatörler

Operatörler Nedir?

Operatörler; basit matematik işlemlerini, mantıksal işlemlerini ve karşılaştırma işlemlerini yapabilceğimiz yapılardır.
Örneğin; Bir hesap makinesi yapıyor olalım. Toplama, çıkarma gibi işlemler için bu operatörlere ihtiyaç duymaktayız...

Atama Operatörü

Atama operatörü, bir değeri bri değişkene atamak için kullanılır. Sembolü eşittir(=) ve atama işlemini, eşittirin sağındaki değeri solundaki değişkene atama şeklinde gerçekleşir.
#include <stdio.h>

int main(){
    int buBirSayidir = 25;
    buBirSayidir = 12

    printf("%d",buBirSayidir);

    return 0;
}

Çıktımız "12" olaraktır. Değişkenimizi ilk olarak 25 değerini atadık. Daha sonra 12 değerine atadık ve çıktımız "12" olmuş oldu.

Aritmatik Operatörler

Aritmatik operatörler, basit matematiksel işlemleri yapabilmezi sağlarlar.

Toplama

Toplama işlemi için artı(+) sembolü kullanılır.
#include <stdio.h>

int main(){
    int x = 10 + 10;
    int y = 4;

    int z = x + y;

    printf("%d",z);

    return 0;
}

Çıktımız "24" olucaktır. X değişkenimizi 10 + 10 işlemi ile 20 değerini atadık, y değişkenimizi 4 değerini atadık. Ardından z değişkenmize ikisinin toplamı olan "24" değerini elde ettik.

Çıkarma

Çıkarma işlemi için eksi(-) sembolü kullanılır.
#include <stdio.h>

int main(){
    int x = 10;
    int y = 5 - 3;

    int z = x - y;

    printf("%d",z);

    return 0;
}

Çıktımız "8" olucaktır. X değişkenmizi 10 değerini atadık. Y değişkenimizi 5 - 3 işleminin sonucu olan 2 değerini atadık. Son olarak x'den y'i çıkardık ve sonucumuz 8 olmuştur.

Çarpma

Çarpma işlemi için yıldız(*) sembolü kullanılır.
#include <stdio.h>

int main(){
    int s1 = 5;
    int s2 = 5;

    int s3 = (3 + 2) + s1 * s2;

    printf("%d",s3);

    return 0;
}

Çıtımız "30" olucaktır. s1 ve s2 değişkenimize 5 değerini atadık. s3 değişkenmizin değeri; işlem önceliğinden 3 + 2 değeri 5 ve s1 ve s2 çarpımı 25'ın toplamının sonucu olan 30 değerini alır.

Bölme

Bölme işlemi için eğik çizgi(/) sembolüdür.
#include <stdio.h>

int main()
{
    int s1 = 4;
    int s2 = 2;

    int sonuc = s1 / s2;

    printf("%d", sonuc);

    return 0;
}

Mod Alma

Mod alma işlemi için yüzdelik(%) sembolü ile yapılır.
Mod alma, bir sayının bir sayıya bölümünden kalan sayıdır.
#include 

int main(){

    int sayi1 = 19;
    int sayi2 = 2;

    int sonuc = sayi1 % sayi2;

    printf("%d", sonuc);

    return 0;
}

Çıktımız "1" olucaktır.

Aritmatik Atama Operatörleri

Aritmatik atama operatörleri, hem matematiksel işlem yapabilen hem de atama işlemini yapabilen operatörlerdir. <degisken1> <Aritmatik Operatör>= <islem> şeklinde yapılır.
#include 

int main(){

    int sayi1 = 3;
    sayi1 += 12; // sayi1'i 12 değeri kadar artırır. Yeni değer, 15

    int sayi2 = 10;
    sayi2 -= 2; // sayi2'den 2 değerini çıkartır. Yeni değer, 2

    int sayi3 = 4;
    sayi3 *= 4; // sayi3 ile 4 değerini çarpmaktadır. Yeni değer, 16

    int sayi4 = 20;
    sayi4 /= 20; // sayi4'ü 20 değerine bölmektedir. Yeni değer, 1

    return 0;
}
