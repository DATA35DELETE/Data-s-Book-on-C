Döngü Yapıları

Döngü Nedir?

Döngü, bir kod bloğunu birden fazla kez tekrar çalıştırılmasına denir.
Döngü Yapıları, bir döngü oluşturarak program akışını değiştirebilirler. Aynı karar yapılarında olduğu gibi program akışında değişiklikler yapabilirler.
Örneğin bir sayaç uygulaması yaptığımızı düşünelim. Her zaman için tek tek printf komutu ile zamanı bildirirsek, belki yüz bin belki milyon satır kod yazmamız gerekebilir. Bu sorunu çözmek için ise döngü yapıları kullanılır. Döngü yapıları sayesinde yüzlerce satır kodluk işi sadece bir kaç satır kod ile yapabilmekteyiz.

While

While, parantezlerindeki değer bir olduğu sürece kod bloğunu çalıştırmaktadır.
Konumuzu pekiştirmek için bir proje yapalım. Projemiz sıfırdan yüze kadar sayıları yazdıran bir program yazalım. Ayrıca While parantezlerinde mantıksal ve karşılaştırma operatörleri kullanılabilir.
#include <stdio.h>

int main(){

    int sayac = 0;

   while (sayac <= 100)
   {
        printf("Dongunun %d. adimi\n", sayac);

        sayac++; // ++, bir arttırma operatörüdür
   }
}

Kodları derleyip çalıştırdığımız da sıfırdan yüze kadar tüm sayılar "Dongunun X. adimi" formatında yazılır.

Kısır Döngü

While döngüsünde kısır döngü tehlikesi vardır. Eğer parantezlerine direk bir değerini yazarsak döngü sonsuza kadar çalışır ve program donanıma fazla yük bindirmesiyle sonuçlanır.
#include 

int main(){

    int sayac = 0;

   while (1)
   {
        printf("Dongunun %d. adimi\n", sayac);

        sayac++;
   }
}

Program derlenip çalıştırıldığında, kaynak kullanımı CPU tarafında %5.4 gibi, bu program için absürt bir kullanıma sahiptir.

For

For, miktarı belli olan ve "i" değişkeni ile döngünün hangi adımında olduğunu anlayabilceğimiz bir döngü yapısıdır.
Şimdi ise While ile yaptığımız örneği For ile yapalım.
#include 

int main(){

   for (int i = 0; i <= 100; i++)
   {
        printf("Dongunun %d. adimi\n", i);
   }
}

Kodları derleyip çalıştırdığımızda sonuç aynıdır. Tek farkları daha az kodla aynı işi yapabilmemizdir.

Break

Break; döngü yapılarında Switch-Case-Default karar grubunda kullandığımız, döngülerde döngüyü "kırmak" yani durdurmak için kullanılan yardımcıdır.
Bir önceki örneğimiz şu şekilde modifiye edelim. Döngü onuncu adıma geldiğinde kırılsın ve döngü son bulsun.
#include <,stdio.h>

int main(){

    // For döngüsü örneği
    for (int i = 0; i <= 100; i++)
    {
        printf("Dongunun %d. adimi\n", i);

        if (i >= 10) // Eğer bir satırlık bir iş varsa
            break;   // kod bloğu açmayabiliriz
    }


    // While döngüsü örneği
    int sayac = 0;

    while (sayac <= 100)
    {
        sayac++;

        printf("Dongunun %d. adimi\n", sayac);

        if (sayac >= 10)
            break;
    }
}

İki döngüde onuncu adımda son bulucaktır.

Continue

Continue, döngüyü geçerli olan döngü adımının atlamasını sağlar.
Pekiştirmek için önceki örneğimzde, on ila doksan arasındaki tüm döngü adımlarını atlasın.
#include 

int main(){

    // For döngüsü örneği
    for (int i = 0; i <= 100; i++)
    {
        if (i >= 10 && i <= 90)
            continue;

        printf("Dongunun %d. adimi\n", i);
    }


    // While döngüsü örneği
    int sayac = 0;

    while (sayac <= 100)
    {
        sayac++;

        if (sayac >= 10 && sayac <= 90)
            continue;

        printf("Dongunun %d. adimi\n", sayac);
    }
}

Kodları derleyip çalıştırdığımız da döngülerin on ila doksan döngü adımlarını atladığını görmekteyiz.
