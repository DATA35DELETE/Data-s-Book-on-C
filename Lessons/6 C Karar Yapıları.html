Karar Yapıları

Karar Yapıları Nedir?

Karar yapıları, verilen koşula göre program akşının yolunu değiştirebilen yapılara denir.
Program akışı, bir programdaki kodların çalışma sırasına denir. Program akışı kod dosyasının en başından en sonuna kadar kodları teker teker çalıştırmaktadır. Karar yapıları ise bu akışın bir kısmını değiştirememizi sağlayan yapılardır. Örneğin; A durumu için z kodlarını, B durumu için de y kodlarını çalıştırabiliriz. İşte bu gibi durumlarda karar yapıları kullanmaktadır.

If-Else If-Else

Bu yapı grubu, karar yapılarınında sıkça kullanılan bir yapı grubudur.

If

If, karar grubunun başıdır. Parantezlerindeki değer bir ise kod bloğundaki kodlar çalışır, sıfır ise kod bloğundaki kodlar çalışmaz.
// Kodlar...

short int kosul1 = 1;
if (kosul1)
{
    // kosul1 ifadesi bir olduğundan bu kısımdaki kodlar çalışıcaktır
}

// Kodlar...

short int kosul2 = 0;
if (kosul2)
{
    // kosul2 ifadesi sıfır olduğundan bu kısımdaki kodlar çalışmıcaktır
}

// Kodlar...
Aynı zamanda operatörlerden karşılaştırma ve mantıksal operatörler ile de kullanımı mümkündür.
// Kodlar...

if (100 > 19)
{
    // İfadenin sonucu 1 olduğundan bu kısımdaki kodlar çalışıcaktır
}

// Kodlar...

if (1 && 0)
{
    // İfadenin sonucu 0 olduğundan bu kısımdaki kodlar çalışmıcaktır
}

// Kodlar...

Else If

Else If, If'den sonra ki karar yapsıdır. Eğer If'in parantezlerindeki ifade sıfır ise program If'in altındaki Else If'lere bakmaya başlar. Ayrıca bir If'ten sonra birden fazla Else If koyulabilir.
Else If ve birden fazla If kullanımın arasındaki fark ise mantık ve kullanım amacıdır. Eğer bir koşulu birden fazla If ile karar edersek, aynı kodların tekrar çalışabilir veya istenmeyen bir zamanda kodların çalıştırılabilir. Eğer Else If kullanırsak program akışı aynı zamanda tek yapıda karar gerçekleştirirken, aynı durum için birden fazla If kullanırsak ise program akışı gereğinden fazla karar yapar ve performans kayıplarına neden olabilir.
O zaman konuyu pekiştirmek için bir örnek yazalım... Kullanıcının sınav sonucunu girip, sınav soncuna göre bir çıktı üreten bir program yazalım.
#include <stdio.h>

int main(){

    int sinav_sonucu = 0;

    printf("Lutfen sinav sonucunuzu giriniz: ");

    scanf("%d", &sinav_sonucu);

    if (sinav_sonucu == 100)
    {
        printf("Sinavdan 100 alarak ustun bir basari gostediniz...");
    }
    else if (sinav_sonucu <= 99 && sinav_sonucu >= 80)
    {
        printf("Sinavda yuksek bir basari gostediniz...");
    }
    else if (sinav_sonucu <= 79 && sinav_sonucu >= 60)
    {
        printf("Sinavda iyi bir basari gostediniz...");
    }
    else if (sinav_sonucu <= 59 && sinav_sonucu >= 40)
    {
        printf("Sinavda ortalama bir basari gostediniz...");
    }
    else if (sinav_sonucu <= 39)
    {
        printf("Sinavda kotu bir basari gostediniz...");
    }

    return 0;
}

Kodları derleyip çalıştırdığımız program çalışıcaktır ve girdiğiniz sınav sonucunuza göre çıktı varklı olucaktır.

Else

Else, bu karar grubunun son yapısıdır. Eğer If ve Else If'ler çalışmaz ise kendi kod bloğu çalıştırılır.
Else, bir If'in alt kısmına sadece bir kere yerleştirebilir. Parantezleri yoktur.
Şimdi ise ufak bir proje daha yapalım. Bir seçim programı yazalım ve kullanıcı seçim ksımında olmayan bir numara girerse, Else ile karar edelim bir mesaj bildirelim.
#include <stdio.h>

int main(){

    printf("0: Cikis\n");
    printf("1: Giris\n");
    printf("2: Kayit Ol\n");

    short int secim = 0;

    printf("--> ");
    scanf("%d", &secim);

    if (secim == 0)
    {
        printf("\nCikis yapiliyor...");
    }
    else if (secim == 1)
    {
        printf("\nGiris yapiliyor...");
    }
    else if (secim == 2)
    {
        printf("\nKayit yapiliyor...");
    }
    else
    {
        printf("\nGecerli olmayan bir secim sectiniz");
    }

    return 0;
}

Kodlayıp derliyelim ve çalıştıralım.

Switch-Case-Default

Bu karar grubu ise tek bir değer karar edilicekse kullanılır. Yazılış şekilleri If-Else If-Else yapısından biraz farklıdır fakat aynı amaç için kullanılır.
Switch'in parantezleri, Case'lerin karar edeceği değişken tanımlanır. Eğer Case'ler çalışmaz ise Default kısmı çalışmaktadır.
Şimdi ise örneğimizi Switch-Case-Default ile yeniden yazalım.
#include <stdio.h>

int main(){

    printf("0: Cikis\n");
    printf("1: Giris\n");
    printf("2: Kayit Ol\n");

    short int secim = 0;

    printf("--> ");
    scanf("%d", &secim);

    switch (secim)
    {
    case 0:
        printf("\nCikis yapiliyor...");
        break;
    case 1:
        printf("\nGiris yapiliyor...");
        break;
    case 2:
        printf("\nKayit yapiliyor...");
        break;
    default:
        printf("\nGecerli olmayan bir secim sectiniz");
        break;
    }

    return 0;
}
