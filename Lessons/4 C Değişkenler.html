Değişkenler

Değişken Nedir?

Değişkenler, programın içersinde bilgi tutan yapılardır.
Değişkenler genelde, daha sonraki zamanlarda gerekll olacak olan bilgileri depolamak için kullanılır. Bu bilgiler genelde geçicidir ve program kapatıldığında bilgiler temizlenir.
Örneğin; Bir banka sisteminde kullanıcı kendi hesabına girmek istesin. Normal şartlarda kullanıcı bir kere şifre girer ve işlemler ardından banka hesabına giriş yapılır. Bu durumu sağlayan yapı ise değişkenlerdir. Kullanıcının girdiği şifre bilgisini bir değişkene atanır ve gerektiği alanlarda değişken çağrılarak bilgi kullanılır. Eğer değişkenler olmasaydı kullanıcı sürekli şifresini girmek zorunda kalıcaktı..

Sayısal Değişkenler

Değişkenlerin türleri vardır. Bunlardan biri ise sayısal değişkenlerdir. Bu tür, tanımlanan değişkenin türü gereği, değeri sayılar ile ifade edilir.

Short

C programlama dilinde en küçük değer tutabilen sayı türüdür. -32,768 ila 32,767 arasında sayı tutabilir.
Düzenleyici bir türdür. Yani tek başına kullanılmak yerine Integer sayı türünün sayı aralığını küçültür.
"unsigned" ile de negatif sayılar görmezden gelinir ve sadece pozitif sayılar alınabilir. 0 ila 65,535 arasında sayı alabilir.
Amacı ise küçük sayıları depolamaktır. Bellekte kapladığı alan 2 bayttır.
short int eksiYuz = -100;
unsigned short int yuz = 100;

Integer

Integer, en çok tercih edilen sayı türüdür. -2,147,483,648 ila 2,147,483,647 arasında sayı tutabilir.
Ana türdür. Düzenleyicilerin düzenlediği yapıdır.
"unsigned" ile de negatif sayılar görmezden gelinir ve sadece pozitif sayılar alınabilir. 0 ila 4,294,967,295 arasında sayı alabilir.
Bellekte kapladığı alan ise 4 bayttır.
int eksiBesMilyon = -5000000;
unsigned int besMilyon = 5000000;

Long

Çok uzun sayılar için kullanılır. -9,223,372,036,854,775,808 ila 9,223,372,036,854,775,807 arasındaki sayıları tutabilir.
Bellekte 8 baytlık bir yer kaplamaktadır.
long long dunyaNufusu = 8000000000;

Float

En çok tercih edilen ondalıklı sayı türüdür.  3.4 × 10^-38 ila 3.4 × 10^38 arasında sayı tutabilir.
Bellekte kapladığı yer ise 4 bayttır.
float ondaklikliSayi = 123123.123123;

Double

Double ise yüksek hassasiyet gerektiren matematik işlemlerinde kullanılır.
Bellekte kapadığı yer ise 8 bayttır.
double karmasikSayi = 2313123.34234e10;

Char

Char değişken tipi, normal şartlarda tek bir harf tutar.
char buyukA = 'A';
        
Tek tırnak içerisine yazılır.
Fakat bu hali pek kullanışlı olmadığından biz dizi şeklinde olan halini kullanıcağız. Dizileri henüz görmedik ama ilerki sayfalarda daha detaylı şeklinde işlemekte olacağız...
char[] hello = "Merhaba, Dünya!";
        
Çift tırnaklar içerisine depolanmak istenen metin yazılır.

Depolama Sınıfları

C programlama dilinde, diğer dillerin aksine erişim belirteçleri yerine depolanma şekilerine göre erişim belirlenir. Aşağıdaki tabloda erişim sınıflarının tablo şeklinde verilmiştir.
İsmi İşlevi Kullanımı
Auto Bulunduğu bloğun içerisinde farklı, bloğunun içerisindeki blokda farklı değerler alabilmesi için kullanılır. auto ismi = değer
Static Nerede ilk tanımlanırsa, bir sonraki tanımları (değerlerini) yok sayan ve üzerine işlem yapıldığında her yerde geçerli olan bir depolama türüdür. static ismi = değer
Extern Bir kod dosyasında tanımlanan değişkeni bir diğer kod dosyasında değer atamayı sağlar. extern ismi = değer

Kullanıcıdan Veri Almak

C programlama dilinde kullanıcıdan konsol üzerinden veri alabilmek için scanf komutu kullanılır.
#include 

int main()
{
    int s1;

    printf("Bir sayi giriniz: ");

    scanf("%d", &s1);

    printf("Girilen sayi: %d", s1);

    return 0;
}

Bu programda s1 adlı değişkene, kullanıcı tarafından girilen sayıyı atıyoruz ve ekranda gösteriyoruz...
Eğer farklı türdeki değişkenler için kullanıcılardan veri alıncak ise scanf tırnaklar içerisindeki yüzdelik(%) işaretinin yanındaki harfi değiştirmeliyiz.Almak istediğimiz veri türleri için, tablodaki kullanımı uygulayabiliriz.
İstenilen Tür Kullanılan Sembol
Integer %d
Long %ld
İşaretsiz(unsigned) Integer %u
Float veya Double %f
Char[] %s

Değişken Türleri Dönüştürme

Bazı durumlarda Float türü gerekirken, elde Integer oluyor. Bu durumu çözmek için değişken türlerini dönüştürürüz...
Aşağıdaki başlıkların altında önemli türlerin birbirleriyle dönüşümleri verilmiştir.

Integer → Float

int s1 = 203;
float s2 = (float) s1;

Float → Integer

float s1 = 123.12;
int s2 = (int) s1;

Noktadan sonraki sayılar silinir.

Integer → Char[]

#include 
#include 

int main()
{
    char yaziSayi[] = "1000";

    int sayi = atoi(yaziSayi);

    printf("'%s' -> %d\n", yaziSayi, sayi);

    return 0;
}

Ekrana hem 1000'i Char[] olarak hem de Integer olarak yazılır.

Char[] → Integer

int sayi = 247593;
char yaziSayi[10];

sprintf(yaziSayi, "%d",sayi);

Sayı ne kadar büyük ise köşeli parantezlere o kadar sayı yazılır.

Sabitlik

Değişkenler yapısı geriği değiştirebilir olarak yapıdadır. Ancak sabit değişkenlerde tanımlanabilir. Bir kere tanımlanır ardından hiç bir zaman değiştirilemezler.
İki tür sabitlik vardır. "const" ve "#define". Aralında fark yoktur. Sadece bellekte farklı şekillerde yer kaplarlar.
#define DEGISKEN_ISMI 12
int const degiskenIsmi = 12;

Typedef

Typedef ise, bir değişken türüne kendi özel ismimizi vermemizi sağlayan bir yapıdır.
Örnek kod aşağıda verilmiştir.
#include 

typedef int tamSayi;

int main()
{
    tamSayi sayi = 16;

    printf("%d", sayi);

    return 0;
}

Program çalıştırılıp çıktımıza bakılırsa "16" görülmektedir.
